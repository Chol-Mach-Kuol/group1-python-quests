# Group 1 - Python Quests

Learn Python by Coding - A collection of Python programming exercises organized by difficulty level.

## Project Structure

```
group1-python-quests/
├── quests/
│   ├── quest_03_treasure_chest.py
│   ├── quest_08_potion_brewer.py
│   ├── quest_13_maze_of_many_choices.py
│   ├── quest_18_loop_of_riddles.py
│   └── quest_25_number_wizard.py
└── README.md
```

## Completed Quests

### Level 1: The Magic of Words and Numbers
- **Quest 3: The Treasure Chest** - Working with data types (integers and strings)

### Level 2: A Dialogue with the Machine
- **Quest 8: The Potion Brewer** - Basic arithmetic operators

### Level 3: The Crossroads of Logic
- **Quest 13: The Maze of Many Choices** - if-elif-else statements

### Level 4: The Power of Repetition
- **Quest 18: The Loop of Riddles** - while loops with user input

### Level 6: The Grand Challenge
- **Quest 25: The Number Wizard** - Enhanced guessing game with feedback

## How to Run

```bash
python3 quests/quest_03_treasure_chest.py
python3 quests/quest_08_potion_brewer.py
python3 quests/quest_13_maze_of_many_choices.py
python3 quests/quest_18_loop_of_riddles.py
python3 quests/quest_25_number_wizard.py
```

## Requirements

- Python 3.x
